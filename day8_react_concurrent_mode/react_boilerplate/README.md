# React boilerplate with webpack dev server

## Dev

```
$ npm install
$ npm run dev
```

> `webpack-dev-server` runs on 4001 with  hot-reloading

## Deploy

```
$ npm install
$ npm run build
```

> builds `bundle.js`

## Components

- webpack 4
- babel 7 with react preset
